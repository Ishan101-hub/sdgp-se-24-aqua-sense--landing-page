# AquaSense – Multi-zone Water Management & Leak Control

A modern, responsive landing page for **AquaSense**, an IoT-powered smart water management system. The system monitors real-time water usage across multiple household sublines, detects leaks instantly, and allows selective shutoff of affected pipelines—all while keeping the rest of the house functional.

## **Features**

* Highlights IoT system functionality: sensors, valves, ESP32, Firebase, and mobile app
* Showcases real-time water usage monitoring, leak detection, and automatic shutoff
* Displays analytics, push notifications, and plumbing service details
* Demonstrates contributions to UN SDG goals: Clean Water (SDG 6), Innovation (SDG 9), Sustainable Communities (SDG 11), Responsible Consumption (SDG 12)
* Responsive design optimized for desktop and mobile

## **Tech Stack**

* **Frontend:** HTML, CSS, JavaScript
* **IoT Integration:** ESP32, Flow Sensors, Solenoid Valves
* **Cloud & Real-time Data:** Firebase Realtime Database

## **Purpose**

This repository contains the **landing page** used to showcase the AquaSense system, its features, SDG impact, and team, providing a clear and professional introduction for users, stakeholders, and potential customers.
